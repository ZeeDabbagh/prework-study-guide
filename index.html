HTML

  • The head element contains information about the webpage
  • The body element represents the visible content shown to the user
  • Elements are everything in the section
  • Tags are what's in between the brackets
  • Parent elements contain everything like the body tag
  • Child elements are the sub elements in the parent

CSS

  • A margin indicates how much space we want around the outside of an element
  • A padding indicates how much space we want around the content inside an element
  • A selector defines the element or attributes to which the rules, or declarations, will apply
  • Declarations contain two important components: the CSS property we want to apply and the value of the property

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branh and switches to it
  • git checkout main: takes us back to the main branch
  • git add -A: adds modification in the current working branch to the staging area - -A flag indications that we want to add all changes
  • git commit -m: adds a commit message that describes what the commit contains

JavaScript

  • A variable is a named container that allows us to store data in our code
  • Control flow is the order in which a computer executes code in a script
  • We must first create the function then CALL the function for it to work
